# Vision Fluxo - Dashboard de Atendimento

Sistema de monitoramento e configuração de fulfillment com integração IA e sincronização SQL Server via PowerShell.

## 🚀 Como subir para o GitHub

Para colocar este projeto no seu GitHub, siga estes comandos no terminal da pasta do projeto:

1. **Inicialize o Git**:
   ```bash
   git init
   ```

2. **Adicione os arquivos**:
   ```bash
   git add .
   ```

3. **Crie o primeiro commit**:
   ```bash
   git commit -m "feat: estrutura inicial com integração Firebase e API de sincronização"
   ```

4. **Conecte ao seu repositório remoto** (Crie um repositório vazio no GitHub primeiro):
   ```bash
   git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
   ```

5. **Envie os arquivos**:
   ```bash
   git push -u origin main
   ```

## 🛠️ Tecnologias
- **Frontend**: Next.js 15, Tailwind CSS, ShadCN UI
- **Backend**: Firebase Firestore, Next.js API Routes
- **IA**: Google Gemini via Genkit
- **Automação**: Script PowerShell customizado

## ⚙️ Configuração IIS
O projeto já contém um arquivo `web.config` na raiz para facilitar o deploy em servidores Windows com IIS e iisnode.
